# agentpack-template-default

Default template for [agentpack](https://github.com/ryoooo/agentpack).

## Contents

- `.devcontainer/` - VS Code devcontainer configuration
- `agentpack.yml` - Manifest skeleton

## Usage

```bash
agentpack init --template github:ryoooo/agentpack-template-default
```

Or use as the default:

```bash
agentpack init
```
